# GTEL Ensaios Elétricos — Etapa 1

Aplicativo Android base para relatórios de ensaio Megger, HIPOT e VLF em campo.

## O que esta Etapa 1 contém

- Projeto Android em Kotlin.
- Jetpack Compose.
- Material Design 3.
- Tema visual GTEL com azul e laranja.
- Navegação entre telas.
- Telas principais criadas.
- Botões de navegação funcionando.
- Room Database configurado inicialmente.
- Lista simples de relatórios salvos.
- Criação e exclusão simples de relatório.
- GitHub Actions para gerar APK.
- Arquivos `gradlew`, `gradlew.bat` e `gradle/wrapper` incluídos.

## O que ainda não foi implementado nesta etapa

- PDF profissional.
- Câmera.
- Assinatura digital.
- Gestão documental completa.
- Banco completo com todos os campos técnicos.
- Relatório fotográfico real.

Essas funções entram nas próximas etapas, depois que o APK da Etapa 1 for validado.

## Como subir no GitHub

1. Crie um repositório novo no GitHub.
2. Envie todos os arquivos deste projeto para a raiz do repositório.
3. Confira se na raiz aparecem:
   - `app/`
   - `.github/`
   - `gradle/`
   - `build.gradle`
   - `settings.gradle`
   - `gradle.properties`
   - `gradlew`
   - `gradlew.bat`
   - `README.md`

## Como gerar o APK pelo GitHub Actions

1. Abra o repositório no GitHub.
2. Clique na aba **Actions**.
3. Clique em **Android Build**.
4. Clique em **Run workflow**.
5. Aguarde finalizar.
6. Baixe o APK em **Artifacts** com o nome:
   `GTEL-Ensaios-Eletricos-Etapa1-APK`.
